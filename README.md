# DecideFlix

# Problema que o sistema resolve

Muitas pessoas gastam mais tempo tentando decidir o que assistir do que realmente assistindo. Isso gera frustração, demora, discussões e até desistência.

# Cliente 

Pessoas passam muito tempo escolhendo filmes/séries têm várias opções, mas não chegam a um consenso
ficam paralisadas pela quantidade de escolha querem decidir algo rapidamente para aproveitar o tempo

# Funcionalidade principal 

O usuário insere 3–5 títulos que está considerando e o sistema gira uma roleta virtual para escolher automaticamente um deles.
Botão principal: “Aceitar e assistir”, levando o usuário a uma página final com a recomendação vencedora.

# Como o sistema entrega valor

- Reduz a indecisão para menos de 30 segundos
- Evita discussões e impasses em grupos
- Transforma uma tarefa estressante (decidir) em algo divertido
- Ajuda casais, famílias e amigos a chegarem a um acordo rápido
- Gamifica o processo de escolha, deixando leve e prático
